What is a pixel?

The atomic unit of every digital image

An image is nothing magical — it is a 2D array (matrix) of pixels. Each pixel stores intensity values as integers from 0 to 255.
A 1920×1080 Full HD image contains 2,073,600 pixels. With 3 channels, that's over 6.2 million numbers.

Coordinate system
Top-left origin, row-major indexing

        

        
Unlike Cartesian math, image coordinates use a top-left origin. NumPy indexes as img[row, col] — reversing the usual x,y order.

        
⚠️
OpenCV uses img[y, x] but many geometry functions take (x, y) tuples. Always check the convention.

        

          
Interactive · Move cursor over the grid

          
          
img[y=—, x=—]   →   pixel coordinates

        

        

          
indexing


# top-left
top_left = img[0, 0]
# bottom-right
h, w = img.shape[:2]
br = img[h-1, w-1]
# region of interest
roi = img[50:150, 100:200]

          
Memory layout
Images are stored in row-major order. Iterating rows is cache-friendly; columns is not.

RGB channels
Three matrices stacked into one 3D array

        

        
A color image is actually three separate grayscale matrices — R, G, B — stacked into shape (H, W, 3).

        

          
Interactive · The same image split across 3 channels

          
          
OriginalR channelG channelB channel

        

        
Critical: OpenCV loads as BGR not RGB. Index 0=Blue, 1=Green, 2=Red.

        
channels


r_channel = img[:, :, 2]  # BGR: index 2 = Red
g_channel = img[:, :, 1]
b_channel = img[:, :, 0]

Color spaces
Different coordinate systems for the same color

        

        
A color space is a mathematical coordinate system for describing colors — each optimized for a different task.

        

          
RGBScreens
R0–255
G0–255
B0–255
Additive model — mixes light.
cv2.COLOR_BGR2RGB

          
HSVColor tracking
H0–179°
S0–255
V0–255
Separates hue from brightness.
cv2.COLOR_BGR2HSV

          
LABColor diff
L0–100
a±128
b±128
Perceptually uniform.
cv2.COLOR_BGR2LAB

          
GrayscaleClassical CV
L0–255
Single channel. Reduces data 3×.
cv2.COLOR_BGR2GRAY
