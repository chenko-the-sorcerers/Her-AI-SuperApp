Machine Learning

Dari regresi linear hingga neural network, pelajari algoritma yang menggerakkan ekosistem AI modern.

8 Modul 50+ Topik AI Lab · Marchel Andrian Shevchenko
01
Foundation

Pengantar Machine Learning

Pahami apa yang sebenarnya dilakukan ML dan mengapa cara berpikirnya berbeda dari pemrograman biasa.

Apa itu Machine Learning?

Perbedaan pemrograman klasik dan ML: dari aturan eksplisit menuju induksi dari data. Tiga paradigma utama adalah supervised, unsupervised, dan reinforcement learning.

SupervisedUnsupervisedReinforcementInductive Learning

Hipotesis, Model & Ruang Fitur

Contoh latih direpresentasikan sebagai vektor fitur. Hipotesis adalah fungsi yang dipelajari dari data, sedangkan model adalah ruang pilihan fungsi yang mungkin.

Feature VectorHypothesisTraining SetInstance Space

VC Dimension & Kapasitas Model

VC dimension mengukur kapasitas model: seberapa kompleks pola yang dapat dipisahkan. Kapasitas tinggi meningkatkan fleksibilitas, tetapi juga risiko overfitting.

VC DimensionShatterModel CapacityGeneralization

Bias-Variance Tradeoff

Overfitting terjadi saat model terlalu kompleks dan menghafal noise. Underfitting muncul saat model terlalu sederhana. Keduanya dikendalikan lewat bias, variance, dan regularisasi.

OverfittingUnderfittingBiasVariance
02
Core Concept

Supervised Learning

Kerangka paling umum dalam ML: belajar dari data berlabel untuk memprediksi label baru.

Komponen Algoritma ML

Setiap algoritma supervised learning memiliki model, fungsi loss, dan prosedur optimisasi. Pilihan ketiganya membentuk inductive bias.

ModelLoss FunctionOptimizationEmpirical Risk

Model Selection & Cross-Validation

Training, validation, dan test set membantu mengukur generalisasi. k-Fold cross-validation memberi estimasi performa yang lebih stabil.

k-Fold CVTrain/Val/TestSRMAICBIC

Generatif vs Diskriminatif

Model generatif memodelkan distribusi data dan kelas, sedangkan model diskriminatif langsung memodelkan batas keputusan atau probabilitas kelas.

GeneratifDiskriminatifParametrikNon-parametrik
03
Algorithm

Regresi & Klasifikasi Dasar

Algoritma fundamental yang tetap relevan untuk baseline, interpretasi, dan data tabular.

Linear & Polynomial Regression

Regresi memodelkan hubungan fitur dan target kontinu. Least squares mencari parameter yang meminimalkan jumlah kuadrat error prediksi.

Least SquaresPseudoinversePolynomialResiduals

Decision Trees & Ensembles

Pohon keputusan mempartisi ruang fitur. Random Forest memakai bagging, sedangkan Gradient Boosting membangun model secara sekuensial.

Information GainGiniRandom ForestXGBoostBagging

k-Nearest Neighbors

k-NN melakukan prediksi berdasarkan mayoritas tetangga terdekat. Sederhana dan non-parametrik, tetapi mahal saat prediksi pada dataset besar.

k-NNEuclidean DistanceLazy LearningCurse of Dimensionality
04
Probabilistic

Probabilistic Models

Menggunakan teori probabilitas untuk membangun model yang mengekspresikan ketidakpastian.

Teori Probabilitas untuk ML

Ekspektasi, variansi, independensi, Bernoulli, Multinomial, dan Gaussian menjadi building blocks untuk model probabilistik.

MLEGaussianBernoulliEkspektasiVariansi

Naive Bayes Classifier

Naive Bayes memakai teorema Bayes dan asumsi independensi kondisional. Cepat, interpretable, dan efektif untuk klasifikasi teks.

Bayes TheoremCond. IndependenceGaussian NBLaplace Smoothing

Bayesian Networks

Bayesian Network memodelkan dependensi antar fitur menggunakan directed acyclic graph, sehingga asumsi independensi dapat dibuat lebih fleksibel.

DAGTANFSSJMutual Information
05
Industry Core

Linear Discriminative Models

Model yang langsung memodelkan batas keputusan, sering menjadi baseline kuat untuk production.

Generalized Linear Model

GLM memperluas regresi linear untuk klasifikasi melalui fungsi aktivasi. Geometrinya adalah hyperplane pemisah di ruang fitur.

HyperplaneDecision BoundaryOne-vs-RestMulti-class

Logistic Regression

Logistic regression menghasilkan probabilitas kelas melalui sigmoid. Cross-entropy, gradient descent, dan regularisasi membuatnya stabil untuk klasifikasi.

SigmoidCross-EntropyGradient DescentL1/L2 RegSGD

Evaluasi Model Klasifikasi

Confusion matrix, precision, recall, F1, ROC curve, dan AUC membantu memilih threshold dan memahami trade-off performa.

Confusion MatrixPrecisionRecallF1ROC-AUC
06
Advanced

Support Vector Machine

Algoritma klasifikasi yang memaksimalkan margin untuk mendorong generalisasi yang lebih baik.

Hard Margin vs Soft Margin

Hard margin menuntut data linearly separable sempurna. Soft margin menambahkan toleransi misklasifikasi melalui parameter C.

MarginSlack VariablesOutlierParameter C

Maximum Margin Hyperplane

SVM mencari hyperplane dengan margin maksimum. Support vectors adalah titik paling kritis yang menentukan batas keputusan.

Quadratic ProgrammingLagrangeSupport VectorsDual Problem

Kernel Trick

Kernel trick memungkinkan SVM menangani data non-linear dengan menghitung kemiripan di ruang fitur berdimensi tinggi secara implisit.

RBF KernelPolynomial KernelMercer TheoremNon-linear
07
Must Know

Neural Networks

Dari perceptron satu neuron hingga deep network berlapis, fondasi arsitektur AI modern.

Single Layer Perceptron

Perceptron menghitung kombinasi linear fitur lalu menerapkan aktivasi. Ia konvergen jika data linearly separable, tetapi tidak mampu memecahkan XOR.

Perceptron RuleStep FunctionConvergenceLinear Separability

Multi-Layer Perceptron

MLP menyusun input, hidden layers, dan output. Aktivasi non-linear seperti ReLU dan sigmoid membuat jaringan mampu memodelkan pola kompleks.

Forward PassReLUSoftmaxHidden Layer

Backpropagation

Backpropagation menghitung gradien loss terhadap bobot secara efisien menggunakan chain rule dari output menuju input.

Chain RuleGradientVanishing GradientBatchNorm

Optimisasi & Regularisasi

SGD, momentum, RMSprop, Adam, dropout, weight decay, dan early stopping membantu training lebih stabil dan generalisasi lebih baik.

AdamSGDDropoutWeight DecayEarly Stopping
08
Unsupervised

Unsupervised Learning

Menemukan struktur tersembunyi dalam data tanpa label melalui clustering, reduksi dimensi, dan density estimation.

K-Means Clustering

K-Means mempartisi data ke K cluster dengan meminimalkan variasi dalam cluster. Pemilihan K dapat memakai elbow method atau silhouette score.

K-MeansK-MedoidsElbow MethodSilhouette

Hierarchical & Density Clustering

Agglomerative clustering membangun dendrogram. DBSCAN menemukan cluster bentuk arbitrer berdasarkan density dan lebih robust terhadap noise.

DendrogramAgglomerativeDBSCANWard Linkage

Gaussian Mixture Models & EM

GMM memodelkan data sebagai campuran distribusi Gaussian. Expectation-Maximization memperbarui soft assignment dan parameter distribusi secara iteratif.

GMMEM AlgorithmSoft ClusteringPCAt-SNE
