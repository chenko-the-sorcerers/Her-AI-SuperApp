01
Foundation

Text Fundamentals

Sebelum menyentuh model apapun, pahami dulu apa itu teks — dan tantangan yang dihadapi mesin saat membacanya.

Tokenization

Langkah pertama setiap pipeline NLP. Memotong dokumen jadi unit kecil (token) — mencakup word-level, subword (BPE, WordPiece, SentencePiece), dan character-level strategies.

BPEWordPieceSentencePieceNLTKspaCy

Basic Preprocessing

Case-folding mengecilkan ukuran vocab. Stop word removal menghilangkan noise frekuensi tinggi. Stemming memotong sufiks. Lemmatization menggunakan vocabulary dan morfologi — lebih lambat tapi lebih akurat.

Case-foldingStop WordsStemmingLemmatizationNLTK

POS & NER

POS tagging menandai peran gramatikal token. Named Entity Recognition (NER) mengekstrak orang, tempat, organisasi. Dependency parsing memetakan hubungan sintaksis.

POS TaggingNERDependency ParsespaCy
02
Core

Text Representations

Mengubah token menjadi representasi numerik — langkah kritis sebelum pemodelan.

Bag-of-Words (BoW)

Representasi paling sederhana: hitung frekuensi kata per dokumen. Kehilangan urutan kata, tapi cepat dan efektif untuk text classification dasar.

VectorizationCountVectorizerCosine Similarity

TF-IDF

Memberikan bobot pada kata — kata yang sering muncul di seluruh dokumen dianggap kurang penting, kata yang spesifik per dokumen lebih berbobot.

Term FrequencyIDFVector Space Model
03
Classical

Classical NLP Models

Naive Bayes, Logistic Regression, SVM — pendekatan klasik yang masih relevan untuk baseline dan production.

Naive Bayes untuk Teks

Asumsi independence antar fitur. Cepat, interpretable, dan sangat efektif untuk spam detection dan sentiment analysis dengan data kecil.

Logistic Regression & SVM

Model linear dengan regularisasi L1/L2. SVM dengan kernel trick untuk decision boundary non-linear. Fondasi untuk memahami model yang lebih kompleks.

04
Deep

Deep Learning for NLP

Word embeddings, dense representations, dan arsitektur neural pertama untuk teks.

Word Embeddings

Word2Vec, GloVe, FastText — mengubah kata menjadi vektor dense yang menangkap hubungan semantik. king - man + woman ≈ queen.

Feed-Forward + CNN untuk Teks

Arsitektur neural sederhana untuk text classification. CNN 1D untuk menangkap n-gram patterns tanpa rekurensi.

05
Sequential

Sequential Models

RNN, LSTM, GRU — arsitektur yang didesain untuk memahami urutan dan konteks temporal dalam teks.

RNN & LSTM

Recurrent Neural Networks memproses teks secara sekuensial. LSTM menambahkan forget gate untuk mengatasi vanishing gradient dan menangkap long-term dependencies.

Encoder-Decoder + Attention

Arsitektur sequence-to-sequence dengan attention mechanism — dasar dari terjemahan mesin modern sebelum era Transformer.

06
Modern

Transformers & Modern NLP

Self-attention, BERT, GPT, fine-tuning, RAG, dan evaluasi model bahasa modern.

Self-Attention & Multi-Head Attention

Mekanisme inti Transformer. Setiap token "melihat" semua token lain dalam sekuens. Scaled dot-product attention.

BERT & Fine-tuning

Pre-training + fine-tuning paradigm. Masked Language Modeling. Transfer learning untuk downstream tasks — klasifikasi, QA, NER.

RAG & Modern Applications

Retrieval-Augmented Generation — menggabungkan LLM dengan knowledge base eksternal. Prompt engineering, chain-of-thought, dan evaluasi output.
